# Jolli Extensions

IDE extensions for Jolli integration.

## Overview

This directory contains IDE extensions that bring Jolli's documentation automation capabilities directly into development environments.

## Extensions

| Extension | Description |
|-----------|-------------|
| [`vscode/`](./vscode/) | VS Code extension for AI-powered documentation generation |

## VS Code Extension

The Jolli VS Code extension provides:

- **1:1 Documentation** - Generate markdown documentation for source code files
- **Local RAG** - Vector database for semantic search over documentation
- **AI Chat Assistant** - Webview-based chat interface with RAG integration
- **File Decorations** - Visual status indicators showing documentation sync state

See [`vscode/README.md`](./vscode/README.md) for detailed documentation.
